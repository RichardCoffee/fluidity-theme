




 Actions
========================
 tcc_custom_css ------------- insert css into inline stylesheet
 tcc_enqueue ---------------- enqueue front-end styles/scripts
 tcc_navbar_signout --------- insert navbar html before signout button
 tcc_register_widgets ------- register widgets extending TCC_Basic_Widget
 tcc_widget_class_loaded ---- load widgets extending TCC_Basic_Widget
 tcc_widget_signout --------- insert widget html before signout button
